1. 参考我的博客 -- [巧妙利用 iOS 的快捷指令配合 GitHub Actions 实现自动化](https://github.com/yihong0618/gitblog/issues/198) 拿到 action id

# GitHub Actions

3. 配置 iOS 的捷径
4. 用捷径触发
